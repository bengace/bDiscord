# bDiscord
A basic Discord bot, used/made only for educational purposes. Written in C# using Visual Studio.

The project uses the following libraries / NuGet packages:

* Discord.Net
* Newtonsoft.Json
* Nito.AsyncEx
* RestSharp
* SuperSocket
* WebSocket4Net
* TwitchLib
* SmartIrc4net

All rights of those go to their original owners.
